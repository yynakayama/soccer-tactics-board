# Soccer Board

サッカー用の戦術ホワイトボードアプリです。ブラウザだけで動きます。

## 開き方

`index.html` をブラウザで開くと使えます。

ローカルサーバーで開く場合:

```powershell
node dev-server.js
```

その後、ブラウザで `http://127.0.0.1:4173/` を開きます。

## できること

- フィールド上の味方・相手選手をドラッグ移動
- 味方選手の背番号と名前登録
- 味方選手の追加、控え管理、選択中の選手との交代
- 相手選手の背番号編集
- ボードリセット時に相手番号を初期化
- 味方登録と配置はブラウザのローカル保存に保存
- 4-4-2、4-3-3、4-2-3-1、3-5-2 の配置反映
